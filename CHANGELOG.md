# Changelog

All notable changes to this project are documented here.

## [1.1.0]
### Added
- Split `index.html` into separate `style.css` and `script.js` files
- Added `favicon.svg`
- Added Open Graph meta tags and `preview.png` for social link previews
- Added MIT `LICENSE`

## [1.0.0]
### Added
- Initial Mandelbulb raymarching WebGL viewer
- Interactive controls: Power, Depth, Hue sliders
- Orbit camera with mouse/touch support
- Soft shadows, ambient occlusion, and filmic tonemapping
